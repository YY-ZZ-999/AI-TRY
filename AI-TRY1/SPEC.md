# 1123 个人网站 - 规格说明文档

## 1. 项目概述

- **项目名称**: 1123 个人网站
- **项目类型**: 单页面个人主页
- **核心功能**: 展示个人心情、要求、学习计划和实习经历
- **目标用户**: 访客、招聘方、朋友

## 2. UI/UX 规格

### 布局结构

- **导航栏**: 固定顶部，包含网站标题和导航链接
- **英雄区域**: 全屏高度，展示个人头像和欢迎信息
- **内容区块**: 垂直滚动，包含以下部分
  - 我的心情 (Mood)
  - 我的要求 (Requirements)
  - 学习计划 (Learning Plan)
  - 实习经历 (Internship)
- **页脚**: 社交链接和版权信息

### 响应式断点

- 移动端: < 768px
- 平板: 768px - 1024px
- 桌面: > 1024px

### 视觉设计

#### 配色方案

- **主色调**: `#1a1a2e` (深蓝黑)
- **次色调**: `#16213e` (深蓝)
- **强调色**: `#e94560` (珊瑚红)
- **辅助色**: `#0f3460` (中蓝)
- **文字色**: `#eaeaea` (浅灰白)
- **次要文字**: `#a0a0a0` (灰色)

#### 字体

- **标题字体**: "Playfair Display", serif
- **正文字体**: "Noto Sans SC", sans-serif
- **数字/装饰**: "Fira Code", monospace

#### 间距系统

- 区块间距: 80px
- 元素间距: 24px
- 内边距: 20px (移动端), 40px (桌面)

#### 视觉效果

- 卡片: 玻璃拟态效果 (backdrop-filter: blur)
- 阴影: 柔和的投影
- 动画: 滚动时的淡入效果
- 悬停: 微妙的放大和颜色变化

### 组件列表

1. **导航栏**: 固定透明背景，滚动后变实色
2. **英雄区**: 全屏背景 + 渐变遮罩 + 居中内容
3. **心情卡片**: 带图标的心情展示
4. **要求列表**: 带序号的要求项
5. **时间线**: 学习计划和实习经历使用垂直时间线
6. **社交链接**: 图标按钮

## 3. 功能规格

### 核心功能

1. **平滑滚动导航**: 点击导航链接平滑滚动到对应区块
2. **滚动动画**: 内容区块进入视口时淡入
3. **响应式布局**: 适配所有屏幕尺寸
4. **交互反馈**: 按钮和卡片悬停效果

### 用户交互

- 导航点击 → 平滑滚动到目标区域
- 卡片悬停 → 轻微上浮 + 阴影加深
- 社交图标悬停 → 颜色变化 + 放大
